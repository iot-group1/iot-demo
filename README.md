# Beacon

Main entry: app.js


Prerequisite:

Os: raspbian

nodejs(npm) 6.11.0

install modules: bluetooth,bluez,libbluetooth-dev,libudev-dev

sudo apt-get install bluetooth bluez libbluetooth-dev libudev-dev

install blean(iBean module)

npm install bleacon
